# Ingredients

- 3 tbsp Hershey's cocoa powder
- 1 cup Lactantia Chocolate milk
- 2-3 tbsp zero calorie sweetener (I used erythritol)
- 1 scoop protein powder of choice (I'm using Optimum Nutrition's double rich chocolate)
- 1 tsp xantham gum
- 4 pcs Oreo thins

# Kitchen Items

- Hand blender
- Ninja Creami

# Preparation

- Add all ingredients except Oreos to a pint
- Use hand blender to combine them
- Freeze until solid through (I usually leave in freezer for >24hr) if not frozen enough, the NInja Creami will melt the icecream when passing through it
- Take out and run Creami in Lite mode
- Add Oreos
- Run Cream in Mixin mode
