# Ingredients

- 3 tbsp Hershey's cocoa powder
- 1 cup Lactantia Chocolate milk
- 2-3 tbsp zero calorie sweetener (I used erythritol)
- 1 scoop protein powder of choice (I'm using Optimum Nutrition's double rich chocolate)
- 0.5 tsp xantham gum
- 4 pcs Oreo thins
- Optional: 1-3 tsp powdered peanut butter - adds nuttiness

# Kitchen Items

- Hand blender (optional)
- Ninja Creami

# Preparation

- Add all ingredients except Oreos to a pint
- Use hand blender to combine them (or just shake them, this is going through the creami so everything will eventually mix, just try to avoid having all the dry ingredients settle to the bottom)
- Freeze until solid through (I usually leave in freezer for >24hr) if not frozen enough, the NInja Creami will melt the icecream when passing through it
- Take out and run Creami in Lite mode
- Add Oreos
- Run Cream in Mixin mode
- Note: This yields more of a frozen yogurt consistency, if you want it thicker i would re-freeze and put it through Lite mode once more
