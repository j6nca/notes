# Background

`thanos-tabs` is a small project sprouted from my constant chaotic mess and collection of browser tabs. A lot of the times I open a page, and think "thats nice, I would like to watch/read more about that later" and then suddenly chrome has slowed to a crawl and i have 100+ tabs open where they've shrunken and I don't even know what's what anymore (thanks for finally rolling out vertical tabs to help alleviate this a bit...).

# What does it do?

This plugin takes your currently open tabs (excluding pinned tabs) and halves them periodically. Just like that 🫰!

You can check out the project on [github](http://github.com/gravityctl/thanos-tabs) or download the extension from the [Chrome store](https://chromewebstore.google.com/detail/thanos-tabs/fapkhicekebpdjgaoeijhdcmfhboolpb).
